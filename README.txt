The SatFlows.py Script calculates Saturation Flows based on a set of of .dis (discarge files) from VISSIM.

Step 1: Paste the .dis files in the Input_DIS_files folder
Step 2: Edit the script in order to change (if required) the parameters. You can edit the script even with NotePad.
	The parameters are located on the upper part of the script
Step 3: Run the script

The output is located in the folder 'Output' under the name Output_SatFlows.csv
